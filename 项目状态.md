## 📝 部署流程（下次部署参考）

### 1. 本地开发
```bash
cd deep-survey-backend
npm install
npm start  # 本地测试
```

### 2. 推送到 GitHub
```bash
git add .
git commit -m "修复/更新内容"
git push
```

### 3. Railway 自动部署
- Railway 检测到推送后自动部署
- 等待 1-2 分钟
- 检查部署日志

### 4. 验证
```bash
# 健康检查
curl https://deep-survey-backend-production.up.railway.app/health

# 提交测试
curl -X POST https://deep-survey-backend-production.up.railway.app/api/submit \
  -H "Content-Type: application/json" \
  -d '{...测试数据...}'
```

### 5. 检查飞书表格
- 打开飞书表格
- 确认数据已写入
